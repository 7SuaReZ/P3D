Pokémon3D - Official Translation Project
========================================

This game is the work of Nilllzz. Visit his tumblr for official
updates on the game here: http://nilllzz.tumblr.com/

Branches are set up like that :
<pre>
--- master [contains ORIGINAL game files]
\
 \
  |--- en
  |
  |--- fr
  |
  |--- [...]
</pre>

Each language must have a specific branch named after its ISO 639-2 code.

You can contribute by forking the repo and sending a pull request,
or you can ask to be a part of the project and become the maintainer
for your language.
